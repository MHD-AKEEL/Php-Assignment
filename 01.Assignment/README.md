
# Job Application Form



## Features

- Secure form handling with input sanitization (htmlspecialchars, trim, stripslashes).

- Fields: Full Name, Email, Contact Number, DOB, Position, Resume (PDF), Cover Letter, LinkedIn, Experience, Skills.

- Validation: Required fields, email format, 10-digit contact, PDF resume, LinkedIn URL, at least one skill.

- Displays errors clearly if validation fails.

- Shows submitted details on success, skills as a comma-separated list.

- Clean layout with CSS styling and user-friendly "Go Back" link.
