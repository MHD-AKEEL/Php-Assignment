
# Grocery Bill Calculation (PHP)




## Features

- Simple HTML form to input quantities for Biscuits, Noodles, Bread, Milk, Eggs, and Dhal.

- PHP input sanitization using a custom input() function.

- Calculates subtotal for each item and total bill.

- Displays total bill on a separate page.

- Includes "Calculate" and "Clear" buttons for user convenience.
